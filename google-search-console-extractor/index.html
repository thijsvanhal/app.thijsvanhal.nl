Google Search Console Extractor

Met de Google Search Console Extractor haal je gemakkelijk data op met door middel van de Google Search Console API.

Hoe werkt de tool?

Vrij simpel. Je logt namelijk in met Google, selecteert een property, datum range én eventuele filter en klikt vevolgens op 'Data ophalen' !

De tool gaat de data voor je ophalen en er wordt automatisch een Excel bestand gedownload met alle data, happy analyzing!

Deze tool is als hobbyproject gemaakt. Heb je een bug gevonden of een idee hoe deze tool verbeterd kan worden? Aarzel niet om contact met mij op te nemen!
